LYNK Electrical Demo v1.2

Cable Tray Support, Clearance & Tag Generator

Download installer (.exe)

Bestandsgrootte: ~2.2 MB · Windows 64-bit · Revit 2024, 2025 & 2026

Melding van je browser?

Je browser kan een waarschuwing tonen omdat het een .exe bestand is. Dit is normaal voor elke software-download.
Chrome: klik op het pijltje naast het bestand → "Behouden"
Edge: klik op "..." → "Behouden"
Het bestand komt rechtstreeks van bimlynk.com en is veilig.

Installatie in 3 stappen

1
Start de installer als administrator

Rechtermuisklik → "Als administrator uitvoeren". Kies je Revit versie(s).

2
Open Revit

Het LYNK Electrical Demo tabblad verschijnt automatisch in het lint.

3
Activeer met je code

Klik op een tool → voer je activatiecode in uit de e-mail. Je hebt 14 dagen toegang.

Wat zit er in de demo?
  • Cable Tray Support — automatisch ophangbeugels plaatsen
  • Cable Tray Clearance — clearance hulls voor clash detection
  • Tag Generator — Conduit ID en Cable Tray ID generators
  • Demo Revit families (.rfa) zitten als bijlage in je e-mail
  • Handleidingen per tool staan ook in de e-mail
